##### 支付订单状态

1. 订单状态说明

    | 值 | 说明 | 备注 |
    | --- | --- | --- |
    | -1 | 用户取消支付 | 用户取消支付后，DragonEx跳转回接入方时传入的订单状态 |
    | 1 | 成功 | |
    | 2 | 失败 | |
    | 3 | 转账中 |若长期处于转账中，说明有异常，请联系DragonEx处理 |
